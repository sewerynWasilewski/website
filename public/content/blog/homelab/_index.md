## What is a homelab?

A homelab is a personal server setup used for learning and experimentation. Mine runs on a second-hand 1U rack server in my apartment, and serves as a playground for everything from Kubernetes clusters to self-hosted services.

:::note
These posts assume basic Linux knowledge but don't require prior experience with virtualization or infrastructure automation.
:::
